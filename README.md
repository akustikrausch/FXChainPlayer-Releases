# FXChainPlayer

**Precision Audio Monitoring with VST3 Effects**

FXChainPlayer is a lean, native 64-bit audio player for Windows with broad format support and a VST3 effect chain for precision monitoring — without DAW overhead.

## Download

Get the latest release from the [Releases](https://github.com/akustikrausch/FXChainPlayer-Releases/releases) page.

## Features

- **30+ Audio Formats** — WAV, MP3, FLAC, OGG, AAC, M4A, WMA, ALAC, AIFF, Opus, W64, DSF/DFF (DSD), APE, WavPack, Musepack, TrueAudio, AC-3, IFF 8SVX, SID, MIDI, MOD/XM/S3M/IT and 20+ tracker formats
- **VST3 Effect Chain** — Up to 8 serial VST3 effects with drag-and-drop reorder, per-slot dry/wet mix, global chain mix
- **WASAPI Output** — Shared and Exclusive mode, low-latency
- **Real-time Visualization** — 6 spectrum modes (FFT, Spectrogram, Phase, VU, LED HiFi, Landscape), GPU-accelerated waveform
- **Gapless Playback** — Seamless track transitions with pre-loaded next track
- **Studio Compare** — Dual-decoder synchronized A/B playback
- **Auto-Update** — In-app update notifications with one-click install

## System Requirements

- Windows 10/11, 64-bit
- ~100 MB disk space
